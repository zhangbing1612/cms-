# 1.markdown是什么
Markdown是一种可以使用普通文本编辑器编写的标记语言，通过简单的标记语法，它可以使普通文本内容具有一定的格式。
# 2.markdown特点
轻量、简单、通用 
# 3.markdown用途
博客 编写说明文档 ppt、word产品文档
